# promt_formater
我想做一个这样的产品应用，我的产品目标客户是海外用户，包括全球用户。我打算部署到 Cloudflare 上面去提供全球服务。

这个产品的功能是做一个转格式工具，比如你复制一段没有格式的 Markdown 文件传上去，就可以把它格式化成 Markdown 格式。你也可以上传调试时遇到的 prompt，它会帮你格式化这个 prompt，方便分析。另外，上传一个 HTML 文件，它就可以把你的 HTML 文件渲染出来。

产品左边是输入框，右边是显示框。用户进入后会有一个大屏选择要转换的格式：Prompt、Markdown 或 HTML。暂时先做这三个功能

我想通过谷歌的广告 AdSense 去打广告赚钱，所以我的产品需要获得足够多的流量。不一定非要做这个功能，你可以帮我分析一下做哪一块会比较有流量
